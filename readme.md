# Bookstore V3.0
Use ES6 modules, [#134](https://github.com/voidove/jug-jug/issues/134)
## Quick start
```bash
# change directory to your app
$ cd bookstore

# install the dependencies with npm
$ npm install

# start
$ npm run dev
```
go to `http://localhost:8080` in your browser, if port `8080` is available.
